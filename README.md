# MomAndPop

## Assembly frontend

* npm install
* gulp desktop

or

* gulp mobile

### Dependencies:

* gulp (npm install -g gulp)

## Starting server (2 realizations)

* cd server/project-mon-and-pop
* npm install
* node server.js
* open http://localhost:10100 (by default)

or

* cd server/FF
* npm install
* node app.js
* open http://localhost:3000 (by default)

### Dependencies

* Should be started mongod on :27017
